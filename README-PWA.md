# PWA 追加手順

このアプリはホーム画面に追加できる Web アプリとして使えます。

## Android での追加方法

1. Chrome でアプリを開きます。
2. 画面右上のメニュー「︙」をタップします。
3. 「ホーム画面に追加」を選びます。
4. 表示された画面で「追加」を押します。
5. 追加後はアプリのアイコンからすぐ開けます。

## iPhone での追加方法

1. Safari でアプリを開きます。
2. 画面下の共有ボタン「□↑」をタップします。
3. 画面下の「ホーム画面に追加」を選びます。
4. 「追加」を押します。
5. ホーム画面にアイコンが作成されます。

## Windows での追加方法

1. Chrome または Edge でアプリを開きます。
2. アドレスバー右側のインストールアイコンを確認します。
3. 表示された「インストール」または「ホーム画面に追加」を選びます。
4. 確認ダイアログで「インストール」を押します。
5. 追加後はアプリとして起動できます。

## 使い方のポイント

- オフラインでも最初に一度表示した画面は開きます。
- Google Apps Script への通信はキャッシュされません。
- 入力途中の内容だけは端末内の localStorage に保存されます。
