# 鮮魚原価・入荷管理

登録済みのデータは Google スプレッドシートに保存されます。Google Apps Script（GAS）をウェブアプリとして公開すると、Android・iPhone・Windows・Macから同じデータを利用できます。

## 1. Googleスプレッドシートを作成する

1. [Google スプレッドシート](https://sheets.google.com/) を開き、新しい空白のスプレッドシートを作成します。
2. スプレッドシートの名前を分かりやすいものに変更します（例: `鮮魚原価・入荷管理`）。
3. このスプレッドシートを、利用するメンバー全員に共有します。

`Products` シートと以下の列は、最初のAPIアクセス時に自動作成されます。手作業で作る必要はありません。

`ID / 登録日時 / 更新日時 / 入荷日 / 優先度 / 産地 / 品名 / 規格 / kg数 / 原価区分 / 原価 / 経費込み原価 / ケース原価 / 1尾（P） / 尾数 / コメント`

## 2. Apps Scriptを貼り付ける

1. 作成したスプレッドシートで、メニューの **拡張機能 → Apps Script** を開きます。
2. 既存の `Code.gs` の内容を削除し、このプロジェクトの [gas/Code.gs](gas/Code.gs) の内容を貼り付けます。
3. 左の「+」からスクリプトを追加し、ファイル名を `api` にします。 [gas/api.gs](gas/api.gs) の内容を貼り付けます。
4. 左のプロジェクト設定からマニフェスト ファイルを表示し、[gas/appsscript.json](gas/appsscript.json) の内容に置き換えます。
5. 保存します。

## 3. ウェブアプリとしてデプロイする

1. Apps Script画面右上の **デプロイ → 新しいデプロイ** を選びます。
2. 種類の選択で **ウェブアプリ** を選びます。
3. 「次のユーザーとして実行」は **自分** にします。
4. 「アクセスできるユーザー」は、利用する全員がアクセスできる設定にします。社内利用なら組織内の全員、Googleアカウントを問わない共有なら全員を選びます。
5. **デプロイ** を押し、Googleの承認画面を完了します。
6. 表示された **ウェブアプリURL** をコピーします。URL末尾が `/exec` のものを使います。

コードを更新した場合は、**デプロイ → デプロイを管理 → 編集 → 新しいバージョン** を選んで再デプロイしてください。URLは通常そのまま使えます。

## 4. config.jsへURLを貼り付ける

[config.js](config.js) の先頭付近にある次の値だけを、コピーしたウェブアプリURLに変更します。

```js
apiUrl: 'PASTE_YOUR_GAS_WEB_APP_URL_HERE',
```

例:

```js
apiUrl: 'https://script.google.com/macros/s/xxxxxxxxxxxxxxxx/exec',
```

## 5. 完成・使い方

`index.html` をブラウザで開くと利用できます。

- 登録するとスプレッドシートへ追加され、一覧・印刷プレビューが自動更新されます。
- 編集と削除は、スプレッドシート上のUUID（ID）を使って処理されます。
- 入力途中の内容だけは、その端末のlocalStorageに一時保存されます。登録が完了すると自動で削除されます。
- 複数人が同時に使う場合も、操作後に一覧または印刷タブを開くと最新の共有データを取得します。

## ファイル構成

```text
gas/
  api.gs            # スプレッドシート操作とJSONレスポンス
  Code.gs           # doGet / doPost と action分岐
  appsscript.json   # GASプロジェクト設定
database.js         # GAS APIとのCRUD通信（下書き保存を除く）
config.js           # GASウェブアプリURLを設定
```
